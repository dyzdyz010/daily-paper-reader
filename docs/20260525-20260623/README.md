## 今日简报（AI）
本周速读 60 篇前沿论文，AI 智能体的长期记忆系统正在被重新定义：记忆不仅是数据库，更是可学习、可溯源的弹性资源。  
最值得关注的方向有两个：一是用隐式潜变量把记忆变成可训练、可分配的连续资源（ElasticMem）；二是用数据溯源技术保证记忆的可靠与可解释（Eywa）。  
普通读者可以这样跟进——下次看到具备长期记忆的 AI 助手，不妨问问它的“记忆”是死数据库还是活知识体，这正在成为区分顶尖与平庸的关键。
