## 摘要
零样本物体目标导航（ZS-OGN）要求具身智能体在没有任何预先训练的情况下探索并定位目标物体。为此，近期方法利用基础模型，但通常依赖静态先验且缺乏适应性，导致重复错误和代价高昂的试错。本文提出一个自进化的ZS-OGN框架，实现持续的测试时改进。具体来说，我们通过从过往轨迹中提取可操作知识来构建智能体规则记忆。然后，提出一种基于上置信界的检索策略，通过平衡语义相关性和历史成功率来选择有效规则。此外，引入一个记忆引导的预反思模块，在行动前预测潜在结果，减少低效探索。大量实验表明，我们的方法优于现有零样本基线，在减少不必要步骤的同时，成功率提高了10.1%。

## Abstract
Zero-Shot Object-Goal Navigation (ZS-OGN) requires embodied agents to explore and locate target objects without any prior training. To this end, recent methods leverage foundation models. But they typically rely on static priors and lack adaptation, which leads to repeated errors and costly trial and error. In this paper, we propose a self-evolving ZS-OGN framework that enables continuous test-time improvement. Specifically, we build an agentic rule memory by extracting actionable knowledge from past trajectories. Then, we propose a retrieval strategy based on upper confidence bound, selecting effective rules by balancing semantic relevance and historical success. In addition, we introduce a memory-guided preflection module that forecasts potential outcomes before action, reducing inefficient exploration. Extensive experiments show that our method outperforms existing zero-shot baselines, achieving a 10.1\% improvement in success rate with fewer unnecessary steps.

# EvolveNav 论文详细总结

## 1. 核心问题与研究动机
- **零样本物体目标导航（ZS-OGN）** 要求智能体在不经任何训练的情况下，在未知环境中寻找指定类别的物体（如“沙发”）。
- 现有方法主要分为两类：
  - **基于训练的方法**：依赖大量环境交互进行强化学习或模仿学习，样本效率低、泛化性差。
  - **训练无关（零样本）方法**：利用 LLM/VLM 等基础模型作为语义规划器，但通常基于**静态先验**，缺乏实时适应能力，导致**重复犯错**（例如被视觉相似物体误导）以及**试错成本高昂**（步数预算有限）。
- 关键挑战：**如何在有限步数预算下，同时实现在线自适应和行动预期**，从而避免低效探索。

## 2. 方法论：自演化规则记忆与预反思机制
核心思想：将导航转化为持续自我改进过程，通过经历中的成功与失败提取可操作的导航规则，并在行动前进行风险预判。

### 2.1 整体框架：EvolveNav
- 由两大模块组成：
  - **回合内预反思探索（Intra-Episode Preflective Exploration）**：在决策时回忆经验规则，评估候选前沿风险。
  - **回合间规则自演化（Inter-Episode Rule Self-Evolution）**：每回合后从轨迹中提炼新规则，并动态管理规则库。

### 2.2 自演化智能体规则记忆
- **规则生成**：无论成功或失败，回合结束后 LLM 回顾轨迹，识别关键决策步，再将局部观察与上下文合成通用的导航规则。
- **语义驱动的功劳分配**：
  - 对每一步计算语义相似度得分 \( S_i = \alpha \cdot S^{room}_i + (1-\alpha) \cdot S^{obj}_i \)。
  - 步骤 \(i\) 的初始功劳 \(C_i = S_i - S_{i-1}\)。
  - 成功/失败采用不同裁剪，经 softmax 归一化后加权求和，得到新规则 \(r\) 的初始支持分数 \(S_r = \sum_i w_i \cdot C_i\)。
- **基于 UCB 的动态记忆管理**：
  - 将规则检索建模为多臂老虎机问题。
  - 规则期望效用 \(\mu_r\) 以动量方式更新：\(\mu_r \leftarrow \eta \cdot \mu_r + (1-\eta) \cdot S_r\)。
  - UCB 分数：\( \text{UCB}(r) = \mu_r + \beta \sqrt{\frac{\ln T}{N_r}} \)，其中 \(T\) 为总回合数，\(N_r\) 为规则被使用次数。
  - 新规则（\(N_r=0\)）初始 UCB 设为无穷大，保证其被验证。
  - 按 UCB 降序排序，取出 top-K 规则注入后续预反思提示。

### 2.3 记忆引导的预反思（Preflection）
- 在每个决策步，获取候选前沿后，**先不要行动**，而是构造结构化提示，包含：
  - 各前沿的几何距离与局部语义观察（可见物体、空间布局）。
  - 从规则库检索的 top-K 经验规则（基于 UCB 选出）。
- LLM 必须对每个候选前沿显式预测失败风险（例如“视觉线索与空间规则冲突”），再综合考虑失败概率和探索收益，选择最优、高风险规避的前沿。
- 基础运动控制器执行移动，但设置了最多 \(H\) 步，确保快速响应新观测并从死路中恢复。

### 2.4 空间建图与低层控制
- 利用 RGB-D 和里程计增量建立 2D 占据栅格地图，提取前沿（已知与未知区域边界）。
- LLM 确定目标坐标后，转化为 2D 局部路径点，由 Fast Marching 等低层控制器导航，避免频繁调用 LLM。

- EvolveNav 在 HM3D 和 MP3D 上均取得**零样本方法中最高 SR 和 SPL**：
