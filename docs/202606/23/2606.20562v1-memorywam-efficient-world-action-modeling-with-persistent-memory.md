## 摘要
现实世界中鲁棒的机器人操控不仅需要理解当前观测，还需要记忆与动力学建模。世界动作模型（WAM）通过联合建模视觉前瞻和基于当前与历史观测的动作，具备这些能力，使其成为机器人操控的一种有前景的范式。然而，现有的WAM面临一个基本权衡：推理高效的方法通常仅依赖于近期观测的有界窗口，因此在非马尔可夫环境中表现挣扎；而保留长历史的方法会带来随序列长度显著增长的时间和空间成本。为了解决这一挑战，我们提出了MemoryWAM，一个具有高效持久记忆的世界动作模型。MemoryWAM采用混合记忆设计，结合了最近帧、事件边界锚帧以及总结长程历史的紧凑要点令牌。定制的注意力机制能够检索详细的短期上下文和压缩的长程上下文，支持记忆依赖的决策，同时减少推理延迟和GPU内存占用。在仿真和现实世界中长时程、记忆依赖的操控任务上，MemoryWAM超越了强大的视觉-语言-动作（VLA）和WAM基线，同时保持了良好的计算效率。

## Abstract
Robust robotic manipulation in the real world requires not only an understanding of the current observation, but also memory and dynamics modeling. World action models (WAMs) possess these capabilities by jointly modeling visual foresight and actions conditioned on both current and historical observations, making them a promising paradigm for robotic manipulation. However, existing WAMs face a fundamental trade-off: methods with efficient inference typically condition only on a bounded window of recent observations and therefore struggle in non-Markovian environments, whereas methods that preserve long histories incur time and space costs that grow substantially with sequence length. To address this challenge, we introduce MemoryWAM, a world action model with efficient persistent memory. MemoryWAM uses a hybrid memory design that combines recent frames, event-boundary anchor frames, and compact gist tokens that summarize long-range history. A tailored attention mechanism enables retrieval of both detailed short-term context and compressed long-term context, supporting memory-dependent decision-making with reduced inference latency and GPU memory usage. Across long-horizon, memory-dependent manipulation tasks in both simulation and the real world, MemoryWAM outperforms strong vision-language-action (VLA) and WAM baselines while maintaining favorable computational efficiency.

- **整体含义**：论文提出 MemoryWAM，旨在借鉴人类混合记忆机制（短时记忆、事件边界记忆、要点记忆），通过紧凑的混合记忆设计，在保留长程依赖的同时大幅降低推理开销，解决现有 WAM 的记忆-效率权衡问题。

### 2. 方法论核心思想与关键细节

- **总体架构**：MemoryWAM 基于 MoT（Mixture-of-Transformers）结构，包含一个视频 DiT（用于动力学建模）和一个动作 DiT（用于动作预测）。视频 DiT 从预训练的视频扩散模型（Wan2.2-TI2V-5B）初始化，动作 DiT 单独训练。推理时无需视频生成，仅通过视频 DiT 单次前向更新 KV 缓存，动作 DiT 在该缓存上进行动作去噪。
- **混合记忆设计**：记忆组件由三部分组成：
  - **短时记忆**：保留最近 \(N_{\text{recent}}\) 帧的完整视觉令牌，用于即时闭环控制，窗口大小固定，复杂度为 \(O(1)\)。
  - **事件边界记忆**：保留任务起始帧（锚帧）的完整视觉令牌，因为初始场景常包含关键指令信息且后续可能被遮挡或超出窗口。
  - **要点记忆**：对每个历史视频帧附加少量可学习的要点令牌（\(M \ll L\)，其中 \(L\) 为每帧视觉令牌数），要点令牌通过注意力对历史帧及其上下文进行压缩。在缓存中，非锚帧、非最近帧的视觉令牌被淘汰，仅保留其对应的要点令牌，从而将长程缓存的存储与注意力复杂度从 \(O(N)\) 降至 \(O(N/d)\)（\(d = L/M\) 为压缩比，文中取 \(d=15\)）。
- **注意力机制**：定制注意力掩码控制信息流：视频帧只能看到其之前的上下文，动作令牌同时关注短时、锚帧和要点记忆三个来源，实现高保真局部上下文与压缩长程历史的融合。
- **训练策略**：采用连续流匹配损失，同时在视频和动作分支上训练。通过按推理时的注意力掩码组织训练序列，并添加噪声增强（Self-forcing）来提升鲁棒性。

### 3. 实验设计

- **仿真实验基准**：使用 RMBench——一个专门评估长时程记忆依赖操控任务的仿真基准，包含 9 个不同记忆复杂度的双臂任务（如“按下按钮”“盖住方块”），每任务使用 50 条专家演示训练，测试 100 次 rollout 的成功率。
- **对比方法**：覆盖三类代表：
  - 直接观测到动作的策略：**π0.5**
  - 高效但无长记忆的 WAM：**FastWAM**
  - 保留全历史的 WAM：**LingBot-VA**
- **真实世界实验**：设计了两个记忆强依赖任务（“贝壳游戏”：跟踪被交换的杯子下的方块；“看后按”：观察数字并按相应次数按按钮），使用 ARX 双臂机器人和 RealSense D455 相机，对比 π0.5、LingBot-VA 和 MemoryWAM。

- **消融实验**：验证混合记忆各组件（锚帧、要点令牌、滑动窗口）的贡献，并与全注意力对比。

### 4. 资源与算力

- **模型规模**：总参数量约 6B（视频 DiT 采用 Wan2.2 预训练权重，动作 DiT 约 1B）。训练时对部分任务利用 RoboTwin 数据集额外预训练（如“观察与捡起”任务），其他任务仅在 50 条演示上微调。

- **实验覆盖**：包括 9 个仿真任务、2 个真实世界任务、若干记忆机制对比、3 种消融变体（移除锚帧、移除要点令牌、移除滑动窗口）以及与全注意力的比较。
- **充分性分析**：实验设计较为全面，不仅横向对比了 VLA 和 WAM 基线，还对效率（延迟、显存）和效果（成功率）做了定量分析。消融实验清晰验证了混合记忆各单元对性能的必要性。不过，训练数据量极小（每任务仅 50 条演示），这种数据效率是否在其他更复杂的场景中仍然成立，未作深入讨论。此外，对比方法中对于全历史 WAM（LingBot-VA）的复现细节（如是否采用相同基座网络）未充分交代，可能影响公平性。

### 6. 主要结论与发现

- MemoryWAM 在 RMBench 上取得了 83.0% 的平均成功率，显著高出仅依赖短时历史的 π0.5（10.4%）和 FastWAM（5.9%），并优于保留全历史的 LingBot-VA（78.2%）。
- 在真实世界的两个记忆依赖任务中，MemoryWAM 同样领先，且其较低推理延迟避免了 LingBot-VA 因延迟造成的观测丢失（如交换杯子时未捕捉到状态变化）。
- 混合记忆在效率上优于 RNN/TTT 等紧凑记忆方案，并且比全注意力更加高效，同时保持甚至略微提升了最终策略性能。
- 消融实验表明，要点记忆对长程依赖最关键，锚帧与滑动窗口提供互补增益，而保留全历史反而可能因信息冗余而稍弱于混合记忆。

### 7. 优点

- **创新性与合理性**：将认知心理学中的混合记忆概念引入 WAM，通过要点令牌压缩实现高效长程记忆，设计新颖且有理论支撑。
- **效率与性能的平衡**：通过 \(O(N/d)\) 的内存和计算开销，在保留持久记忆的同时实现了远低于全历史方法的延迟和显存占用。
- **训练-推理一致性**：训练时的注意力掩码与推理时的 KV 缓存结构对齐，避免了分布偏移，并且使用噪声增强缓解了学生强制问题。
- **实验的全面性**：包含仿真、真机、效率测试、消融实验等多维度评估，结果具有说服力。

### 8. 不足与局限

- **语义理解与推理不足**：MemoryWAM 继承了视频扩散模型在高层语义理解和符号推理上的局限，可能难以处理需要抽象逻辑的任务。
- **训练数据量极小**：尽管在小样本下效果不错，但可能受益于基座模型的强先验，其对大规模、高多样性任务的泛化能力尚待验证。
- **对比公平性存在疑问**：与 LingBot-VA 等对比时，是否使用相同预训练模型与训练数据未完全明确；部分任务额外加入了动作历史或预训练，可能干扰对比结果。
- **超参数敏感性**：要点令牌的数量、滑动窗口大小、锚帧数目等关键参数如何影响性能与效率，论文未给出系统分析，实际部署时可能需要大量调参。
- **硬件依赖**：尽管推理延迟降低，但仍然依赖高端 GPU 进行 50 步去噪（仿真），对于低成本边缘设备上的实时应用尚有距离。
