## 摘要
部署在真实场景中的机器人将在其部署过程中积累许多会话和任务的经验。机器人的任务可能经常需要它记住多个会话之前的信息，这使得长上下文记忆对现实部署至关重要。然而，当前大多数机器人记忆基准测试都基于单次情节或短上下文。为了衡量当前机器人记忆系统在更长会话和更多干扰下的表现，我们引入了基于 RoboMME 构建的跨会话基准测试 RoboMME-Interference。对于每个查询情节，我们使用查询相关的先前演示，后跟可控数量的无关会话来构建会话历史，将其作为记忆提供给 VLA 并测量准确度。通过该基准测试直接运行 RoboMME 已发布的记忆增强 $π_{0.5}$ 变体，我们发现，虽然感知记忆变体在提供无任何干扰的历史记录时能提高成功率，但随着无关会话的积累，它们的表现会持续显著下降。通过这一发布，我们强调了长上下文记忆和抗干扰鲁棒性的重要性，并表明当前系统在这些能力上大多表现不佳。项目页面、视频、代码和数据位于 https://robotmemorybench.com。

## Abstract
Robots deployed in realistic settings will accumulate experience across many sessions and tasks over their deployment. The robot's tasks may often require it to remember information from multiple sessions ago, making long-context robot memory important for real-world deployments. However, most robot-memory benchmarks today are based on single episodes or a short context. To measure how current robot memory systems perform on longer sessions with more distractions, we introduce RoboMME-Interference, a cross-session benchmark built on RoboMME. For each query episode, we construct a session history using the query's relevant prior demonstration followed by a controlled number of unrelated sessions, which we provide to the VLA as memory and measure accuracy. Running RoboMME's released memory-augmented $π_{0.5}$ variants unmodified through this benchmark, we find that while perceptual memory variants improve success when given the history without any distractors, they decay strongly and steadily as unrelated sessions accumulate. With this release, we emphasize the importance of long-context memory and robustness to interference and show that current systems largely fail on such capabilities. The project page, videos, code, and data are at https://robotmemorybench.com.

## 6. 主要结论与发现
- **纯净记忆提升显著，但极易被无关干扰破坏**：当历史仅包含相关演示（k=0）时，感知记忆变体能大幅提升成功率（如 FrameSamp‑Modulator 从 18.2% 跃升至 45.3%），然而随着无关会话增加，增益几乎线性衰减，在 k=7 时回落至接近无记忆基线（约 19%）。
- **记忆整合机制差异巨大**：在相同感知表征下，Modulator 机制远优于 Context 和 Expert（FrameSamp‑Modulator k=0 达 45.3%，而 Context 版仅 26.7%），TokenDrop 也有类似规律，说明记忆进入策略的方式比记忆内容本身更能决定抗干扰能力。
- **循环记忆变体在跨会话场景下基本失效**：Recurrent‑TTT‑Expert 和 Context 在所有历史条件下的成功率均 ≤18%，几乎不见记忆带来的任何提升，与无历史基线持平。
- **任务与难度影响不一**：不同任务族的记忆依赖度和天花板差异大；记忆增益主要集中在 easy 和 medium 难度情节上，hard 情节各系统的表现始终贴近地板，表明难点并非仅靠记忆可解。
- **当前机器人记忆系统不具备可靠的长上下文抗干扰回忆能力**，这一问题与 LLM 长记忆领域十分相似，为未来研究指明了方向。

## 7. 优点
- **首次系统性地将“跨会话干扰”引入机器人记忆评测**，弥补了现有基准单会话、短上下文的空白。
- **基准设计严谨且可复现**：公开代码、数据、已发布检查点，干扰单元长度固定、来源跨族，控制无关变量的同时避免矛盾信息。
- **评测覆盖全面**：9 种系统 × 9 个任务族 × 5 种历史条件，兼顾整体衰减趋势、任务维度差异与难度分层分析。
- **突出应用意义**：强调真实部署中信息被大量无关交互淹没的常态，对开发鲁棒长上下文机器人记忆有直接推动价值。

## 8. 不足与局限
- **仅限于仿真，缺乏真实硬件验证**，实际部署中的感知噪声与动态干扰尚未涉及。
- **单检查点单次评估**，未控制训练随机性或运行间变异性，结论可能对特定种子或检查点敏感。
- **覆盖任务不完整**：只能评测先验演示与执行可分离的 9 个 RoboMME 任务，其余 7 个内置线索的任务无法适用，限制了方法在某些记忆类型上的通用性。
- **干扰构造限定为不同族无关会话**，未涵盖来自同族的矛盾信息干扰，也未探索更符合现实的渐进、交错式干扰。
- **未发布逐步骤动作轨迹和完整帧数据**，虽然提供了元数据，但完全可验证性尚有提升空间。
- **算力细节缺失**，使得估算部署成本或在不同规模策略间比较能耗变得困难。
