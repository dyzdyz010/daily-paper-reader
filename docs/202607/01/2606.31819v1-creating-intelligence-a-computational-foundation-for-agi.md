## 摘要
本文介绍了一种基于集合论和超维计算的新型心灵计算理论。传统神经网络依赖于连续权重和矩阵乘法，而该框架处理的是稀疏二进制数据。它将信息表示为离散集合，直接模拟生物神经元群体编码。我表明，联想记忆可以从具有组合扩展隐藏层的网络拓扑结构中自然涌现。学习由拓扑可塑性驱动，而非标量权重调整。此架构将自联想和异联想学习统一在一个核心算法下：通过子集模式匹配和精确最近邻搜索进行信息检索。这些机制以常数时间复杂度运行，在没有连续瓶颈的情况下，桥接了感知数据（稀疏分布式表征）和符号（稀疏全息表征）。将此框架映射到神经解剖学，我提出小脑和新皮层都实现了该算法的变体，使得子集模式匹配成为认知的基本引擎。由于它依赖于离散逻辑而非矩阵算术，该算法可直接转化为内存内硬件。这为具有人类级能效的合成智能开辟了一条新途径。

## Abstract
This work introduces a new computational theory of mind grounded in set theory and hyperdimensional computing. Whereas traditional neural networks rely on continuous weights and matrix multiplication, this framework works with sparse binary data. It represents information as discrete sets, directly modeling biological neural population codes. I demonstrate that associative memory emerges naturally from network topologies featuring a combinatorially expanded hidden layer. Learning is driven by topological plasticity rather than scalar weight adjustments. This architecture unifies auto-associative and hetero-associative learning under a single core algorithm: information retrieval via subset pattern matching and exact nearest-neighbor search. Operating with constant-time complexity, these mechanisms bridge perceptual data (sparse distributed representations) and symbols (sparse holographic representations) without continuous bottlenecks. Mapping this framework to neuroanatomy, I propose that both the cerebellum and the neocortex implement variants of this algorithm, making subset pattern matching the fundamental engine of cognition. Because it relies on discrete logic rather than matrix arithmetic, this algorithm translates directly into in-memory hardware. This opens a new route toward synthetic intelligence with human-level energy efficiency.

## 2. 方法论

### 核心思想
- **信息表示**：  
  全部采用稀疏二进制集，对应神经元的群体编码。分为两类：  
  - **稀疏全息表征（SHR）**：随机生成的小集，所有子集代表同一符号，用于符号、指针、组合结构。  
  - **稀疏分布式表征（SDR）**：元素对应语义特征，用于感知数据编码。
- **联想记忆的本质**：  
  联想行为源于网络的**拓扑结构**：一个组合扩展的隐藏层，每个隐藏节点至少接收两个来自输入的固定连接，并向输出层形成可开关的动态连接。学习就是接通或断开这些连接（拓扑可塑性），而非调整连续权重。检索通过子集匹配和 kWTA（k 个赢家通吃）实现。
- **统一算法**：  
  同一核心算法同时支持异联想（`A → B`）和自联想（`A → A`），仅通过是否为网络提供第二训练通道和输出层维度是否与输入相同来区分。自联想使用内递归，异联想为前馈。

### 关键技术细节与算法流程

- **存储异联想 `A → B`**：
  1. 对输入集 `A` 中所有元素对 `(i,j)` 进行组合扩展，映射到记忆集 `M_ij`。  
  2. 将输出集 `B` 与当前 `M_ij` 做集合并（OR），写入记忆：  
     `M_ij ← M_ij ∪ B`。  
  （复杂度 O(|A|²)，与已存数据量无关）

- **检索（核心子集匹配算法）**：
  1. 令初始匹配集 `X = A`。  
  2. 若 `|X| < T`（模式匹配阈值）返回空。  
  3. 展开 `X`的元素对，聚集对应记忆集内容，得到总响应多重集 `R` 和每元素贡献 `R_i`。  
  4. 对 `R` 做 kWTA（以阈值 `T(T-1)/2` 作为最低激活次数），得到候选输出 `Y`。  
  5. 若 `|Y| < P_B` 返回空。  
  6. 计算每元素的权重 `w_i = |R_i ∩ Y|`。  
  7. 若所有 `w_i ≥ (|X|-1) |Y|` 则收敛，输出 `Y`；否则，剔除最低贡献元素，`X ← X \ X_min`，回到步骤2。  
  重复迭代，整个过程 O(|A|²)，与已存数据量无关。

- **自联想与更新规则**：  
  自联想记忆可搭配多种更新规则（替换、合并、交集、残差、补集、对称差），通过集运算改变内部状态，从而表现出模式补全、去噪、新颖检测、预测生成等多样动力学行为。
- **其他操作**：  
  定义了集上的束（并集）、块编码（不交并集）、多重集聚合、随机子采样、排列等操作，以支撑组合式知识表示、迭代分解、图游走、短时记忆等。

### 方法上的特点
- 不需要矩阵乘法和连续权重。
- 记忆位置就是二值存储（位），容量由维度、稀疏度决定，最优工作密度约50%。
- 模式匹配阈值自动计算，确保在满容量存储随机SHR时，误匹配期望低于1。
- 算法具有抗噪、完成缺失模式、区分已知未知的能力。

- 手写数字分类达到约96%的测试准确率，未经超参优化，仅作为功能验证。未直接与 CNN 等深度模型在同等条件下精细 benchmark 对比，而是旨在展示该系统有处理传统机器学习任务的能力。
- 知识推理实验属定性演示，无标准量化指标对比。

## 6. 主要结论与发现

- **联想记忆的拓扑基础**：  
  离散联想记忆可从具有 O(N²) 隐藏节点的网络拓扑中涌现，无需连续权重；学习通过二值连接开关实现（拓扑可塑性）。
- **统一算法与常数时间**：  
  且一个核心子集匹配算法同时处理自联想和异联想，检索与存储的复杂度为 O(|A|²)，与已存数据总量无关。
- **感知与符号的无缝桥接**：  
  SDR（感知）和 SHR（符号）均为集合，异联想编码器将感知聚类为稳定符号。
- **可映射的神经结构**：  
  小脑的异联想回路和新皮质的自联想柱状结构都可解释为该算法的实例。
- **支持高容量与抗噪**：  
  理论容量大，例如 N=1000、P=10 的自联想可存约96万个模式；检索能抗加性噪声和缺失，能可靠区分已知与未知。
- **适合存内计算**：  
  全二进制逻辑可直接在 DRAM 或 NAND 中实现，实现类似脑的能效（并行超维字，低频运行，功耗约为传统计算机的1/10000）。
- **启发性对应**：子集匹配阈值形成阶梯式反应、束分解极限约为7±2（米勒数），提出可能与短时记忆、认知调节、童年遗忘等现象相关。

## 7. 优点

- **原创性高**：提出了一套完全基于集合和二值操作的实用计算理论，与深度学习范式区分开。
- **简洁优雅**：核心算法仅50行代码，却统一了多种联想记忆功能。
- **生物合理性**：符合脉冲神经网络、群体编码、局部学习、稀疏活跃、预测处理等特性；给出小脑和新皮层实现的可测试假说。
- **理论与实践结合**：既有数学推导（容量、阈值），又有交互示例和C参考实现。
- **硬件友好**：指向后GPU存内计算，极大降低功耗。
- **框架全面**：包含了知识表示、图形、序列、推理等多个认知层面的方案。

## 8. 不足与局限

- **大规模评估缺乏**：除MNIST（96%）外，无大型真实数据集（如ImageNet、自然语言任务）上的性能报告，难以判断其在复杂任务上的竞争力。
- **与现代方法的对比不足**：未与当代向量数据库、图神经网络、记忆增强 Transformer、符号-神经网络等基线进行定量对比。
- **容量和检索的理论假设**：假设模式为随机 SHR 且独立，但实际场景中的相关性可能显著降低有效容量，对此讨论较少。
- **超参数敏感性**：算法表现依赖维度 N、种群数 P 和阈值 T 的选择，且 T 的自动计算基于理想随机模型，实际编码时可能需手动调整。
- **序列记忆与动态推理**：虽然提供了储备池、置换指针链等方案，但未展示长时间依赖、复杂推理上的鲁棒性。
- **实验生态：** 多采用小规模示意或定性案例；支撑 AGI 所需的 scaling 实验缺失。
- **潜在偏差风险**：作者自定指标，缺乏社区公认基准，难以评估泛化性。
- **实现层面限制**：当前为同步数据流，可能不适应异步脉冲驱动，未探讨在线学习中的遗忘策略如何影响实际应用表现。
